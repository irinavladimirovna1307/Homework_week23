# Тарифные карточки - React компонент | React + Vite

Проект реализует адаптивные тарифные карточки с возможностью выбора и визуальным выделением специального тарифа. Создан с использованием React и SCSS.

## Особенности
- Четыре цветовых варианта тарифов
- Адаптивный дизайн
- Интерактивные элементы
- Специальное выделение тарифа 550р:
  - Анимация пульсации
  - Яркая градиентная рамка
  - Эффект "поднятия"
- Оптимизированная верстка

## Технологии
- React 18
- SCSS модули
- CSS анимации
- Flexbox layout

4. Установите зависимости:
